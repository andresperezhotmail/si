## Requirements

- Node 8x or 10x LTS
- GIT for Clone the RESPO

## Install via GIT

```bash
git clone https://github.com/mskian/sys-info.git
cd sys-info
npm install
```

- Execute the Script

```bash
node system.js
```

## Used as CLI

- Open Cloned Location

```bash
npm link
```

```bash
mysysinfo
```

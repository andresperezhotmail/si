## Requirements

- Node 8x or 10x LTS
- GIT for Clone the RESPO

## Install via GIT

```bash
git clone https://github.com/mskian/sys-info.git
cd sys-info
npm install
```

- Execute the Script

```bash
node system.js
```

OR

```bash
npm start
```

## Used as CLI

- Open Cloned Location

```bash
npm link
```

```bash
mysysinfo
```

## Install via NPM

```bash
npm install -g sys-info
```

```bash
mysysinfo
```
